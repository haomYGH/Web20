# react + react-cli + react-redux + react-router-dom + axios + antd

> 项目功能说明:

 1. 采用了react搭建
 2. 使用了react-router-dom来实现路由，实现单页面应用
 3. 使用react-redux做状态管理工具
 4. axios做请求工具
 5. antd做ui框架。
 6. 项目实现了动态路由，跨域配置，请求拦截，路由拦截，状态本地存储等。
 <!-- 7. 项目加入了`px2rem,lib-flexible`实现的移动端适配方案 -->
```
 /*当前目录命令行输入下面的命令*/
    /*安装依赖*/
    npm install
    /* 项目启动 */
    npm start 
    /* 打包 */
    npm run build
```
