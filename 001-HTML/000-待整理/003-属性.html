HTML 标签可以拥有属性。属性提供了有关 HTML 元素的更多的信息。

        属性总是以名称/值对的形式出现，比如：name="value"。

        属性总是在 HTML 元素的开始标签中规定。

        *属性：属性无处不在。

        *全局属性：有style、class、data-*、title属性等等。(详情请查看官网)

        *局部属性：每种元素都能规定自己的属性，这种属性称为：局部属性。(style样式中又有color属性)

        ## 特殊标签拥有特有的属性

        *href="#"  注意： 标签中必须提供 href 属性或 name 属性。
